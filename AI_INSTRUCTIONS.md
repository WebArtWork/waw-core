waw-core is the foundational system module of the waw framework that provides the core CLI entrypoints and a small set of runtime helpers attached to the shared `waw` context, while keeping `cli.js` as a thin dispatcher into `util.*` files; it implements project scaffolding (`waw new` using terminal prompts and `git.forceSync`), module scaffolding (`waw add` using the default module template scaffold), CSS switching for supported front-end project types via `waw.projectType` (`waw css` selecting a repo and syncing into the scss folder), module synchronization (`waw sync` to force-sync modules from their `module.json` repos or publish changes back with hygiene enforcement), basic git workflows through an explicit cross-platform git utility (`forceSync/attach/publish/remove/ensureHygiene`), and PM2 process management (`start/stop/restart`); at runtime it only augments `waw` with stable helpers like `wait(ms)` and `http(hostname, port)` and does not define any application logic, routing, data models, or framework behavior, leaving all higher-level functionality to other modules.
