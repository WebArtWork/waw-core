waw-core is the foundational system module of the waw framework responsible for providing core CLI commands and attaching minimal runtime helpers to the global waw object. It implements project and module scaffolding, CSS switching, module synchronization, PM2 process management, and basic git workflows, while keeping the CLI layer thin and delegating all behavior to small utility functions. At runtime, waw-core only augments the execution context with a few stable helpers (such as wait and http) and does not define application logic, routing, data models, or frameworks, ensuring that all higher-level behavior is introduced by other modules.
